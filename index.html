Cheat Atom


ctrl + p = interface de Atom (les dossier)

ctrl + f = recherche de fichier (Echape/sortir)

ctrl + q = quitter

ctrl + s = Sauvegarder

ctrl + impecr = aller au debut du texte

ctrl + fin = aller tout en bas

ctrl + g + numero = aller a la ligne choisie

ctrl + l = selectionne toute la ligne

ctrl + a = selectionner tout le texte

ctrl + shift + d = duplique la ligne

ctrl + shift + k = supprime toute la ligne ou le curseur se trouve

ctrl + z =annule la dernier action

ctrl + / = mettre la phrase en commentaire

ctrl + maj + h= push, commit , etc a partir d'atom
